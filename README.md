Utilities Macaddress
==========================

A utility to query the macaddress.io api for MAC address information. Obtains
the company name for the specified mac address.

Requirements
------------

* Ubuntu LTS Bionic

Usage
-----
```bash
~$ python mac_utils.py \
>  --token $YOUR_MAC_IO_TOKEN \
>  --macaddress $YOUR_MAC_TO_LOOKUP
```
